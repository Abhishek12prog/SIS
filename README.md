# Student Information System (SIS)

## Project Overview
The **Student Information System (SIS)** is an academic project developed for **ICFAI University**.  
This system is designed to manage student-related information such as student details, academic records, course details, and administrative operations in a simple and efficient way.

The project also includes an **Admin Login** module, which allows the administrator to securely manage student data and system records.

---

## Objectives
- To maintain student records digitally
- To reduce manual paperwork
- To provide secure access for administrators
- To manage student and course information efficiently
- To make updating and viewing records easier

---

## Features
- Admin login authentication
- Secure admin access
- Add student details
- Update student details
- Delete student records
- View student information
- Manage course details
- Store data in an organized database
- User-friendly interface

---

## Modules
### 1. Admin Module
The Admin Module is the main controlling part of the system.  
It allows the administrator to log in securely and manage all student-related information.

**Functions of Admin Module:**
- Admin login
- Add new student records
- Edit student details
- Delete student details
- View all student records
- Manage academic and course information

### 2. Student Information Module
This module stores and displays student-related details such as:
- Student name
- Roll number / ID
- Course name
- Branch / Department
- Contact details
- Academic records
